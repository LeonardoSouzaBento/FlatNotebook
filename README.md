## FlatNotebook (ideal)

FlatNotebook permite criar notas em um formato específico: usando blocos colapsáveis para organizar as informações. No FlatNotebook a hierarquia de títulos é automática e predefinida para você não precisar definir títulos manualmente. Cada bloco tem um nível hierárquico. Você adiciona blocos pai, blocos filhos, blocos netos e assim por diante. Além de textos corridos você pode inserir imagens, tabelas, blocos de código, citações e listas.

## Funcionalidades

- Criação, edição, exclusão, busca e organização de notas
- Formatação de texto automática
- Inclusão e edição de imagens nas notas
- Exportação e importação de notas
- Sincronização entre dispositivos
- Modo offline
- Modo noturno
- Compartilhamento de notas
- Colaboração em notas
- Histórico de versões
- Backup automático
- Autenticação de dois fatores
- Suporte a múltiplos idiomas
